Installation and steps to run:

1) Unzip the file
2) Open cmd in the same folder and run 'node index.js'
3) Now open 'localhost:3000' on the browser.



Tools and Technology Used:
1) NodeJS
2) Bootstrap
3) AJAX



Description:
User can enter any query for books regarding the book itself. It may be bookname or author name and relevant results will be displayed.
